# AYSO Region 76 - Design Tokens Summary for Brizy

## Color Palette

### Primary Blue
**Note**: The main brand color is **orange**.

- **Primary Brand Color (Orange)**: `#ef832f`
  - This is the dominant color used throughout the site for headings, buttons, and CTAs
- **Teal/Blue-Green (closest to blue)**: `#206B70`
  - Used only in dropdown menu sub-headers

### Secondary Red
**Note**: The site does not use a red color as a secondary color.

- **Secondary Accent (Brown/Orange)**: `#D69048`
  - Used for footer, info banner, and dropdown links
- **Light Orange (hover state)**: `#fcc04d`
  - Used for button hover states

### Backgrounds
- **Primary Background**: `#ffffff` (white)
- **Off-White Background**: `#fefefe`
- **Light Grey Background**: `#f2f5f6`
- **Alternate Grey**: `#f0f3f5`
- **Dark Background**: `#2d2f30` (dark grey)

### Text Colors
- **Body Text**: `#1b2b34` (dark grey-blue)
- **Heading Text**: `#06121b` (near black) or `#ef832f` (orange for h1-h3)
- **White Text**: `#fefefe`
- **Muted Text**: `#5b5e5f` (grey)

---

## Typography

### Font Families
- **Heading Font**: `'Fjalla One'` (sans-serif)
- **Body Font**: `'Noto Sans'`

### Base Typography
- **Base Font Size**: `16px`
- **Base Line Height**: `32px` (2rem)

### Heading Sizes
- **H1**: `38.8px` (2.425em)
- **H2**: `22.2px` (1.3875em)
- **H3**: `15px` (0.938em)

## Buttons

### Border Radius
**Border Radius**: `6px`

### Padding Values
- **Standard Button Padding**: `8px 12px` (vertical horizontal)
